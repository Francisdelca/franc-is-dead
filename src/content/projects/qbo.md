description: Una experiencia web para visualizar productos personalizados sobre madera antes de solicitar una cotización.
summary: Editor visual por capas, simulación de materiales y CMS propio para convertir un producto difícil de imaginar en una propuesta tangible.
participation: Diseñé y desarrollé la experiencia completa desde cero, incluyendo la dirección visual, el personalizador de productos, el flujo de cotización y un CMS para el equipo.
technologies:
  - astro
  - cloudflare
responsibilities:
  - Dirección visual y experiencia de producto
  - Editor de fotografías, textos, decoraciones e iconos
  - Integración de enlaces y códigos de Spotify
  - Sistema de capas, máscaras y filtros
  - Flujo de descarga y cotización por WhatsApp
  - CMS visual personalizado
  - Despliegue con Cloudflare, R2 y KV
challenges:
  - El producto es altamente personalizado y las personas necesitaban ver un resultado creíble antes de cotizar.
  - Las vetas de la madera cambian el aspecto final de la impresión y podían generar expectativas incorrectas.
  - No tuve acceso a archivos originales de marca ni a fotografías de alta resolución.
solution:
  - Construí un editor donde cada persona puede cargar su imagen, añadir contenido y adaptar la composición al formato físico.
  - Desarrollé filtros y capas para simular de forma realista la interacción entre la fotografía y las vetas del pino.
  - Reconstruí los recursos visuales necesarios y preparé el contenido público disponible para sostener una presentación consistente.
results:
  - Una demo funcional que permite visualizar, descargar y enviar una personalización para cotizar.
  - Una experiencia artesanal y clásica con una presentación contemporánea y premium.
  - Un sistema administrable construido sin depender de un CMS externo.
